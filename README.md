# ⚡ Energy Monitoring System

Sistema web para monitoramento de consumo de energia com dashboard interativo.

## 🚀 Acesse o projeto
👉 https://energy-monitoring-rki8.onrender.com

## 🧠 Funcionalidades
- 📊 Dashboard com gráfico de consumo
- ⚠️ Alertas de alto consumo
- ➕ Cadastro de consumo (modo demonstração)
- 🔓 Acesso sem login (demo)

## 🛠️ Tecnologias utilizadas
- Python (Flask)
- SQLite
- HTML / CSS
- Chart.js
- Render (deploy)
