## 📌 Sobre o Projeto

Este projeto foi desenvolvido para facilitar o acompanhamento de gastos energéticos. Ele combina o poder do **Python (Flask)** no backend com a fluidez do **Chart.js** no frontend, permitindo uma visualização clara através de alertas e gráficos dinâmicos.

### 🧠 Funcionalidades Principais

* 📊 **Dashboard em Tempo Real:** Gráficos interativos para análise de tendências de consumo.
* ⚠️ **Alertas Inteligentes:** Notificações visuais automáticas para picos de consumo elevado.
* ➕ **Gerenciamento de Dados:** Cadastro simplificado de leituras (modo demonstração habilitado).
* 📱 **Design Responsivo:** Interface adaptável para diferentes tamanhos de tela.
